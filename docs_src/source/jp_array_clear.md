# Array クラスの clear インターフェイス

このページでは`Array`クラスの`clear`メソッドのインターフェイスについて説明します。

## インターフェイス概要

`clear`メソッドは配列の値を空にします。

## 基本的な使い方

`clear`メソッドは呼び出しに特に引数を必要としません。

```py
# runnable
import apysc as ap

ap.Stage()
arr: ap.Array = ap.Array([10, 20, 30])
arr.clear()
assert arr == []
```
