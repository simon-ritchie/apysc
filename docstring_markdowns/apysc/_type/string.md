## `String` class docstring

String class for apysc library.<hr>

**[Notes]**

The `Str` class is the alias of `String`.<hr>

**[Examples]**

```py
>>> import apysc as ap
>>> _ = ap.Stage()
>>> string: ap.String = ap.String("Hello")
>>> string
String("Hello")

>>> string += " World!"
>>> string
String("Hello World!")

>>> string.value = "World!"
>>> string
String("World!")

>>> string.value = "Hello!"
>>> string *= 3
>>> string
String("Hello!Hello!Hello!")
```
