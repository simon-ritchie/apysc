String クラスの概要¶

Stringクラスはapyscの文字列用のクラスです。このクラスは以下のコード例のようにコンストラクタの引数にstrもしくはString型の値を受け付けます:

# runnable
import apysc as ap

string_1: ap.String = ap.String('Hello')
assert string_1 == 'Hello'

string_2: ap.String = ap.String(string_1)
assert string_2 == 'Hello'
