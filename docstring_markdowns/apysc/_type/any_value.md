## `AnyValue` class docstring

Class implementation of any value (a value that can't determine type).<hr>

**[Examples]**

```py
>>> import apysc as ap
>>> any_value: ap.AnyValue = ap.AnyValue(10)
>>> any_value.value
10

>>> any_value.value = 20
>>> any_value.value
20
```
